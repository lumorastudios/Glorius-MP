## About

**Glorius-MP** is a fork of **PocketMine-MP**, following the discontinuation of its official development. Glorius-MP adds multi-version support for Minecraft Bedrock **1.26.0–1.26.30** and will continue to receive updates for future protocol versions.

Glorius-MP is fully compatible with **PocketMine-MP API 5.0.0** plugins, as no API changes or breaking modifications have been made.
